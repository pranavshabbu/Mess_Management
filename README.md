1. Make a connection.php file following the format in "connection_file_example.txt" to connect to your database.
2. Install XAMPP.
3. Copy the Mess_Management folder to 'C:\xampp\htdocs'.
4. Run XAMPP and start the Apache and MySQL modules.
5. Create a database in phpMyAdmin and add a user.
6. Run the website by navigating to 'localhost/Mess_Management/mess_mgmt.html' in a new tab.
